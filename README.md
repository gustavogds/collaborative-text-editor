## Como usar o CLI

```bash
insert <position> <character> # Insere o caractere na posição inserida
```

```bash
delete <position> # Deleta caractere na posição
```

```bash
show # Mostra o estado atual do documento
```

```bash
peers # Mostra os peers conectados
```

```bash
exit # Sai do programa
```
